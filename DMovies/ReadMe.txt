In case of "Microsoft.ACE.OLEDB.12.0 provider is not registered on the local machine" Error

     download and install Microsoft Acces Database Engine

          https://www.microsoft.com/en-us/download/Confirmation.aspx?ID=13255

     Do not change the path
     Keep it 			C:\program Files (x86)\MicrosoftOffice\

Then, in Microsoft Visual Studio change the Solution platform to x86 and debug

     In the Server Explorer, Data Connections change the path (if necessary) and refresh the D_DataBase.accdb

     In the Solution Explorer refresh the D_Movie
